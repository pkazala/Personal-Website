Projects

In this section I have included all of my important projects, which include games done in pygame, websites, and every other type of application that I have done and decided it was worth showing to the world.

  • PY.GAME Project

    A simple game created with the py.game library for Python. It was a surprise project for my girlfriend which is the reason for the graphic desing. The game itself has very simple mechanics where the goal is to fly in-between the pink flowers that appear randomly along the way and gain points the further you go.

  • To-Do Web App created with React

    One of my very first projects built with React JS. The app allows the used to add to-do's or remainders. It makes use of some of the main React hooks such as useState as well as useEffect. On top of that it is connected to Firebase where all to-do's are stored.

  • Discord "random facts" bot

    In this project I decided to build a simple bot for my brothers Discord server which whenever anyone would type "ciekawostka" (polish for "fact") it would respond with a random fact. For the facts I found an API named "numbersapi" which returns random facts about numbers in JSON form. I deployed the bot itself on repl.it which allowed me to create a web server that would keep the bot running even if I closed the tab.
